# Сервис заказа товаров для розничных сетей

Проект представляет собой backend-сервис на Django, реализующий функционал интернет-магазина.  
Пользователь может зарегистрироваться, подтвердить e-mail, просматривать товары, управлять корзиной и оформлять заказ.

## Основной функционал

- Регистрация пользователя с подтверждением по e-mail  
- Авторизация и выход из системы  
- Просмотр списка товаров (каталога)  
- Управление корзиной (добавление, просмотр, удаление)  
- Создание и управление контактами (адреса доставки)  
- Подтверждение заказа с уведомлением по e-mail  
- Просмотр истории заказов  

---

## Установка и запуск проекта

1. Установите зависимости:
pip install -r requirements.txt

2. Выполните миграции базы данных:
python manage.py migrate

3. Запустите сервер разработки:
python manage.py runserver

4. После запуска проект будет доступен по адресу:
http://127.0.0.1:8000/
